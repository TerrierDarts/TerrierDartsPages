### OBS Set-Up

Using OBS create a browser source.

Add the dimensions (**Width:500 / Height:500**).

Paste in the Link:

```
https://terrierdarts.co.uk/obs_links/chat_dots/index.html
```

### Notes

When a person chats the Brown Circle will change colour and move to where the pink circle is.
