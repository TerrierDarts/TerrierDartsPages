### OBS Set-Up

Using OBS create a browser source.

Add the dimensions (**Width:1920 / Height:1080**)

Paste in the Link:

```
https://terrierdarts.pages.dev/obs_links/star_wars_credits/index.html
```

### Notes
Here is the font you will need to download for this to work.

Font Needed : https://www.dafont.com/sf-distant-galaxy.font 

The Font used is `SF Distant Galaxy Alternate.ttf` so take care installing.
