### OBS Set-Up

Using OBS create a browser source.

Add the dimensions (**Width:990 / Height:450**)

Paste in the Link:

```
https://terrierdarts.pages.dev/obs_links/valheim_alerts/index.html
```
Tick `Allow OBS to control the audio.`
