### OBS Set-Up

Using OBS create a browser source

Add the dimensions (**Width:1920 / Height:1080**) 

Paste in the Link: 

```
https://terrierdarts.pages.dev/obs_links/dark_souls_alerts/index.html
```

Tick `Allow OBS to control the audio.`
