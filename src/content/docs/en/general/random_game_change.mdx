### Configuration

In terms of Configuration this has 3 modes, `allgames`, `pggames` and `plus18`. 
|Mode (type)|Description|
|---|---|
|`allgames`| This will mean you can pick from any game that is on the list|
|`pggames`| This will only select a PG Friendly Game Title|
|`plus18`| This will only select a 18+ game title|

The argument `%randomGame%` is populated and this is the game that we will be changing to.
